# 📚 Livraria Online

Uma aplicação web de e-commerce para venda de livros, permitindo que usuários naveguem por catálogos, pesquisem títulos, adicionem ao carrinho e realizem compras de forma simples e segura.

## 🚀 Funcionalidades

* 📖 Listagem de livros com categorias
* 🔎 Busca por título, autor ou gênero
* 🛒 Carrinho de compras
* 👤 Cadastro e login de usuários
* 💳 Sistema de checkout (simulado ou integrado)
* ⭐ Avaliações e comentários de livros
* 📦 Histórico de pedidos

## 🛠️ Tecnologias Utilizadas

* **Frontend:** React / Vue / HTML + CSS + JavaScript
* **Backend:** Node.js / Django / Spring Boot
* **Banco de Dados:** MongoDB / PostgreSQL / MySQL
* **Autenticação:** JWT / OAuth
* **Pagamentos:** Stripe / PayPal (opcional)

## ⚙️ Instalação e Execução

### Pré-requisitos

* Node.js instalado
* Gerenciador de pacotes (npm ou yarn)
* Banco de dados configurado

### Passos

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/livraria-online.git
```

2. Acesse o diretório:

```bash
cd livraria-online
```

3. Instale as dependências:

```bash
npm install
```

4. Configure as variáveis de ambiente:

```bash
cp .env.example .env
```

5. Execute o projeto:

```bash
npm run dev
```
